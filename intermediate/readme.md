* ## intermediate
1. 프로젝트 구현
    * 컴포넌트 구조
      * TodoHeader
      * TodoInput
      * TodoList
      * TodoFooter
    * localStorage 활용

   * 각 컴포넌트에서 localStorage에 데이터 저장/수정/삭제

   * App.vue(container)에서만 데이터를 관리하고 다른 컴포넌트에서는 화면만 조작하도록 변경
   * CSS 개선 (Vuetify 적용)
3. 사용자 경험 개선
   * 모달 컴포넌트 등록
   * 트렌지션 적용
